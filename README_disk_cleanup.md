# 磁盘清理记录

这不是长期配置，只记录这台机器已经确认过的清理结论。

## 第一原则

- Windows 更新或重装后，先看 `C:\Windows.old`
- 不要优先折腾 `Recent`
- 清理 `Windows.old` 用系统自带的 `磁盘清理`

## 已确认

- `Windows.old` 才是大头
- `Recent` 主要是历史记录，释放空间很小

## 下次如果再出现同样情况

1. 确认不需要回退旧系统
2. 确认不需要从 `Windows.old` 恢复文件
3. 以管理员身份打开 `磁盘清理`
4. 选择 `C:` 盘
5. 点击 `清理系统文件`
6. 勾选 `以前的 Windows 安装`
7. 执行清理
